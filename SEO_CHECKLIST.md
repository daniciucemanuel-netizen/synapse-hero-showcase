# Norel Studio — Post-Launch SEO Checklist

## Immediate (Day 1)

- [ ] Set up [Google Search Console](https://search.google.com/search-console) and verify `norelstudio.com`
- [ ] Submit `https://norelstudio.com/sitemap.xml` in Search Console
- [ ] Request indexing for `/`, `/work`, `/blog`
- [ ] Set up [Google Analytics 4](https://analytics.google.com/) and add the tracking snippet
- [ ] Verify OG images render correctly on [LinkedIn Post Inspector](https://www.linkedin.com/post-inspector/), [X Card Validator](https://cards-dev.twitter.com/validator), and [Facebook Debugger](https://developers.facebook.com/tools/debug/)

## Week 1

- [ ] Create and optimize [Google Business Profile](https://business.google.com/) for "Norel Studio"
- [ ] Monitor crawl errors in Search Console
- [ ] Check Core Web Vitals in Search Console → Experience tab
- [ ] Ensure all pages are indexed (Coverage report)

## Ongoing

- [ ] Track rankings for target keywords:
  - "AI growth marketing agency"
  - "content marketing agency"
  - "SEO agency for startups"
  - "brand strategy agency"
  - "digital marketing agency"
- [ ] Publish 2–4 blog posts per month targeting long-tail keywords
- [ ] Build backlinks through guest posts, partnerships, and case studies
- [ ] Monitor and improve Core Web Vitals scores
- [ ] Update sitemap.xml when adding new pages
