| Section padding | `py-32` | `py-20` |

| Section title | `text-4xl md:text-5xl` | `text-3xl md:text-4xl` |
| Row padding | `py-8 md:py-10` | `py-5 md:py-6` |
| Client name | `text-4xl md:text-6xl` | `text-2xl md:text-4xl` |
| Expanded panel padding | `p-8 md:p-10` | `p-6 md:p-8` |
| Expanded panel bottom | `pb-8 md:pb-10` | `pb-5 md:pb-6` |
| Arrow icon | `w-5 h-5` | `w-4 h-4` |

Same layout, same interactions, just tighter and more refined.
